# Dictionary-Py

Dictionary for quick searches via terminal written in python

## Installation


```bash
$ git clone https://github.com/adrianObel/Dictionary-py
$ cd Dictionary-py
$ python setup.py install
```

## How to use

```bash
$ dictpy [language] [word]
```
## Languages soported

 * English
 * Spanish
 * Italian
